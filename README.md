# Basic Implementation of RAG
> A minimal and understandable implementation of Retrieval-Augmented Generation (RAG).

This project provides a foundational implementation of a Retrieval-Augmented Generation system. It is designed to be easy to understand and extend.

## Features
- Document parsing and chunking
- Text embeddings generation
- Vector storage and retrieval
- LLM generation with retrieved context
